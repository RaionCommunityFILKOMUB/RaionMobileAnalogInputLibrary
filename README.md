## Build Link
[Link]: https://github.com/jmsrsd/MobileInputAnalog/raw/master/Build/Raion.MobileInputAnalog.unitypackage

## Usage
By only writing this statement
```
MobileInputAnalog.GetInstance();
```
all the library code will run.
